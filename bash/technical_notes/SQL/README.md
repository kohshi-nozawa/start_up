## SELECT文
```sql
SELECT [取得したい要素] FROM [使用テーブル];
```
### 特定要素だけ表示する
特定の要素だけ表示したい場合、下記のように記述する
**※「game_characterテーブル」から「name」だけ取り出したい場合**
```sql
SELECT name FROM game_character;
```
結果
```
+------+
| name |
+------+
| 山田 |
| 鈴木 |
| 田中 |
| 高橋 |
| 渡辺 |
+------+
```

### 特定要素の複数指定
**※「名前」と「レベル」のみを取得・表示したい場合**
```sql
SELECT name,level FROM game_character;
```
